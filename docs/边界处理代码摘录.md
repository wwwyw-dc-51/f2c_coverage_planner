## 2. 预处理流程 (第1358-1510行)

```
原始 polygon + holes → Cell(外环 + 内环) → Cells
  → Headland 分解流程:
    1. const_hl.generateHeadlands(cells, hl_ratio * robot_width) → mid_hl (头land多边形)
    2. BoustrophedonDecomp.decompose(mid_hl) → sub-cells
    3. const_hl.generateHeadlands(decomp, no_hl_ratio * robot_width) → no_hl (内区)
    4. swath_generator.generateSwaths(angle, spacing, cell) → swaths
```

Cell 创建:
```cpp
f2c::types::Cell cell;
// 外环: polygon 点 → LinearRing
for (point in polygon) { ring.addPoint( Point(x,y) ); }
cell.addRing(ring);
// 内环 (孔洞): holes → LinearRings → cell.addRing(hole_ring)
f2c::types::Cells cells;
cells.addGeometry(cell);
```

分解角度优化: 遍历 5 个候选角度, 选 swath 数 + 丢失数×3 最小的方案

## 3. 候选 Headland 扫描 (第1663-1719行)

9 组 headland 比值:
```cpp
const double hl_candidates[] = {0.30, 0.25, 0.22, 0.20, 0.18, 0.16, 0.15, 0.14, 0.12};

for (double hl_r : hl_candidates) {
    // 1. 生成 headland (缩小后的外边界)
    auto cand_hl = simplifyCells(
        const_hl.generateHeadlands(cells, hl_r * robot_width_));
    // 2. 分解
    auto decomp = boustrophedon_decomp.decompose(cand_hl);
    // 3. 内区 = 再缩小一次 (no_hl)
    auto target = simplifyCells(
        const_hl.generateHeadlands(decomp, no_hl_width_ratio_ * robot_width_));
    // 4. 在 target 上生成 swaths
    // 5. 评分: rate + hl_r*0.03 - sbc.sizeTotal()*0.002
    //    覆盖率越高越好, headland 越宽越好(安全), swath 越少越好(效率)
}
// 按评分排序, 从高到低尝试
```

**核心问题**: Fields2Cover 的 `generateHeadlands()` 对所有多边形做均匀外缩, 不区分瓶颈/宽口。

## 7. simplifyRing / simplifyCells (第567-618行)

去除 OGR buffer 产生的小角度共线冗余顶点:
```cpp
simplifyRing(ring):
    for each vertex:
        sin_angle = |cross(prev→curr, curr→next)| / (len1*len2)
        if sin_angle > sin(0.5°): keep vertex
        else: discard (共线)
    return simplified ring (>=4点, 闭合)

simplifyCells(cells):
    for each cell: simplifyRing(外环) + simplifyRing(各内环)
```

## 8. adjustSwathEndpoints (第487-533行)

对单个 Swath 的起止点做平移:

```cpp
f2c::types::Swath adjustSwathEndpoints(const Swath& swath, double distance) const
{
    if (distance == 0.0) return swath;

    start = swath.startPoint();
    end   = swath.endPoint();
    dx = end.x - start.x;
    dy = end.y - start.y;
    length = sqrt(dx*dx + dy*dy);

    if (length <= 2.0 * abs(distance)) return swath;  // 太短不调

    unit = (dx,dy) / length;

    // distance > 0: start向end移, end向start移 (收缩)
    // distance < 0: start背向end移, end背向start移 (延伸)
    new_start = (start.x + distance * unit.x, start.y + distance * unit.y);
    new_end   = (end.x   - distance * unit.x, end.y   - distance * unit.y);

    LineString new_path(new_start, new_end);
    return Swath(new_path, swath.getWidth(), swath.getId(), swath.getType());
}
```

for each sub_cell in decomp_mid_hl:

## 10. 几何覆盖率估算 (coverage_evaluator.hpp 第341-357行)

```cpp
pair<double,double> computeCoverageGeometric(Swaths& swaths, double net_area, double cov_width)
{
    if (net_area <= 0) return {0,0};

    swath_total = 0;
    for each swath:
        swath_total += distance(swath.startPoint(), swath.endPoint());

    covered = swath_total * cov_width;      // 简化: 矩形面积和
    rate    = min(1.0, covered / net_area);
    return {covered, rate};
}
```

## 12. 完整管线概览 (planCoveragePath 函数, 第1281行起)

```
输入: geometry_msgs::Polygon (ROS消息) + hole polygons
  │
  ├→ 1. 构造 F2C Cell (外环 + 内环/孔洞)
  │
  ├→ 2. 多角度 decompose 优化 (遍历候选角, 选 swath 最少的)
  │
  ├→ 3. 生成 mid_hl (first headland, 默认 0.15*robot_width)
  │     Boustrophedon 分解
  │     自适应 no_hl (窄 cell 单独降侵蚀)
  │     过滤微小 cell / Cell合并(关闭)
  │
  ├→ 4. 候选 Headland 扫描 (9组 0.12~0.30)
  │     对每组: generateHeadlands → decompose → generateHeadlands → swaths → 评分
  │     按评分排序
  │
  ├→ 5. 遍历候选 (从高到低)
  │     ├→ genRoute (OR-Tools TSP + 头land环连接)
  │     ├→ Swath 端点调整 (边界swath减半, 内部正常)
  │     ├→ planDirectPath (direct，Route控制点保真) / F2C PathPlanning (其他模式)
  │     ├→ 边界检查 (6000+段, 原始cells.isPointIn)
  │     ├→ PASSED → RDP简化 → break
  │     └→ REJECTED → continue (换下一个候选)
  │
  ├→ 6. 掉头可行性检查 (非direct模式)
  │
  ├→ 7. 发布可视化 (RViz markers)
  │
  └→ 8. 评估报告 (覆盖率 + 效率 + 综合评分)
```
