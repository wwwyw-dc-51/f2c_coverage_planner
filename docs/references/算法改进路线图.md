## 一、头地转弯类型优化（Headland Turn Types）

### 当前代码
使用 Dubins 曲线做 swath 间的连接，不区分转弯类型，且不约束边界。

### 改进方向：根据头地宽度自动选最优转弯类型

| 转弯类型 | 所需头地宽度 | 耗时 | 适用场景 |
|----------|-------------|------|---------|
| 灯泡转弯 (Bulb) | 大 | 最短 | 头地宽裕 |
| 钩子转弯 (Hook) | 中 | 中 | 转弯半径 > swath间距/2 |
| U型转弯 | 中 | 中 | swath间距大 |
| 鱼尾转弯 (Fishtail) | 小 | 长（需倒车） | 头地狭窄、嵌入式室内场景 |

**决策逻辑**（Jin & Tang 2010）：
```
if headland_width 足够 and r <= w/2 → Bulb转弯（最短前向）
elif headland_width 中等 and r > w/2 → Hook转弯
elif headland_width 受限 → Fishtail转弯（允许倒车）
elif swath间距大 → U型转弯
```

**参考文献**：
- Jin, J. & Tang, L. (2010). *Optimal Coverage Path Planning for Arable Farming on 2D Surfaces.* Transactions of the ASABE.
- Tu, X. & Tang, L. (2019). *Headland Turning Optimisation for Agricultural Vehicles and Those with Towed Implements.* J. Agriculture and Food Research.

## 二、TSP 路径排序优化

### 当前代码
使用 Fields2Cover 的 RoutePlannerBase，内部已集成 OR-Tools 求解 TSP。`ortools_exact_solve` 参数控制求解器日志输出（2026-07-08 确认）。

### 改进方向 A：带转弯代价的 TSP（Krupke 2024）

七步算法，将覆盖率路径规划转化为 LP + 匹配 + 环路优化，最优性差距 ~10-15%。

核心创新：在 TSP 排序中显式建模转弯代价，而不仅考虑距离。

```
当前：TSP(swaths) → 只优化距离
改进：TSP(swaths, turn_costs) → 同时优化距离 + 转弯角度
```

**对嵌入式的适用性**：太复杂（LP 求解 + 大邻域搜索），不适合在线运行。可做离线预处理。

### 改进方向 B：双图方法（WUR 论文）

构建两个图：
- SPG（最短路径图）：建模 headland 上各 swath 端点之间的通行代价
- CPG（覆盖路径图）：swath 内部的代价为零，swath 之间的代价来自 SPG

然后 OR-Tools 求解 O(N²) 级别。

**对嵌入式的适用性**：中等。双图可在离线构建，在线只查表。

**参考文献**：
- Krupke, D. (2024). *Near-Optimal Coverage Path Planning with Turn Costs.* ALENEX 2024. https://arxiv.org/abs/2310.20340
- WUR thesis: Two-graph method for transition cost in headland CPP. https://edepot.wur.nl/669513

## 三、分解角度优化（Decomposition Angle）

### 当前代码
`decomposition_angle = 0.0`，固定不变。

### 改进方向：MSA（Minimum Sum of Altitudes）

定理（Huang 2001）：最优分解方向一定垂直于多边形某条边。

```
候选分解角 = 边缘垂直方向 = 边缘方向 + 90°
对每个候选角：Boustrophedon分解 → 计算各cell swath条数之和 → 选最小的
```

**与 swath 角度优化的关系**：
- 分解角度的候选和 swath 角度的候选来自**同一组边缘方向**
- 两者只差 90°，可共享同一次边缘提取
- 分解角度可离线做一次，swath 角度在线跑

**参考文献**：
- Huang, W.H. (2001). *Optimal line-sweep-based decompositions for coverage algorithms.* ICRA 2001.

## 四、相邻 Cell 合并（Cell Merging）

### 问题
Boustrophedon 分解产生的相邻 cell，如果 swath 方向相似（差 < 30°），独立走会多一次边界掉头。

### 改进方向
```
分解后：
  cell_1 (swath 0°) ─┬─ 方向差 5° → 合并为一个 swath 区块
  cell_2 (swath 5°)  ┘
  cell_3 (swath 85°) ──── 方向差 85° → 保持独立
```

合并条件：相邻 + 角度差 < 阈值 + 合并后不为非凸

**参考文献**：
- Agarwal & Akella (2022). *Area Coverage with Multiple Capacitated Robots.* IEEE RA-L.
