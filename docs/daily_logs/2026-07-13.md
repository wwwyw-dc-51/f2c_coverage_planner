## Phase 2: Snake 优化 5 项改动

### 2A: 斜边 Cell 独立 swath 角度
- **新增函数** `detectSlantedBoundaryAngle()`（~85行）
- 检测 cell 是否贴近斜边界（与 sweep 方向夹角 15°~75°）
- 若是则返回斜边方向作为 swath 角度 → 减少末端三角缝隙
- 垂直边被过滤（避免水平条带中做垂直 swath）

### 2B: fillBoundaryGaps 放宽角度容差
- `angle_tol`：0.996 (cos 5°) → 0.9397 (cos 20°)
- 效果：更多斜边被识别为"平行"，边界补填覆盖率提升

### 2C: RemArea 间隙填补
- 评估后用 `uncovered_grid` 做空间分箱加速聚类
- 5000点上采样限制 + 3×3邻域 bin 搜索（O(n²)→O(n)）
- 对聚类生成 bbox 填充 swath，裁剪后追加到路径

### 2D: 斜边感知 Cell 合并
- 孔洞隔离检查新增"物理贴近"判定
- 若两 cell 边界距离 < 0.5×cov_width → 即使质心线穿洞也允许合并
- 防止 sweep 分解在斜边界产生孤立三角 cell

### 2E: Snake 安全路径（零出界保证）
- Snake 模式跳过 TSP genRoute，手动构建 Route
- Cells 已按圆形绕洞排序 → 逐 cell 拼接 swaths + 直连
- 空 cell 安全检查，找下一个非空 cell 再连接
- **注意**：`swaths_by_cells.at(ci).back()` 必须保证 cell 非空
