## 版本概述

v10.0 是稳定版本，在 v9.12-stable 基础上累积了覆盖率指标统一、TSP 确定性优化、微型连接去冗余、审计工具完善等改动。**21/21 GATE PASS**，全场景有效覆盖率 ≥99%。

## 核心改动

### 算法

| 改动 | 说明 |
|:---|:---|
| **PATH_CHEAPEST_ARC TSP** | OR-Tools 首解策略从 AUTOMATIC 改为确定性贪心，减少随机波动 |
| **微型去冗余** | 安全检查后移除不必要的 U 形绕行连接 |
| **覆盖率指标统一** | raw/effective coverage 明确区分，门禁使用 effective |
| **回滚 mergeCollinearSwathsAcrossCells** | N11_whole 59% 回归修复 |

### 基础设施

| 改动 | 说明 |
|:---|:---|
| **审计工具完善** | 批测报告审计脚本升级 |
| **N3 重复测试** | `n3_repeat_test.sh` 验证 TSP 方差 |

### 贪心算法

| 改动 | 说明 |
|:---|:---|
| **恢复 v9.12 贪心** | 4 variant + 孔洞极角排序 + DP |
